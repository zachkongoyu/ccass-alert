# CCASS Alert

"CCASS Alert" is a Python program for detecting changes in CCASS shares.

## Installation

Use the package manager [git](https://git-scm.com/) to install "CCASS Alert".

```bash
git clone https://github.com/zachkongoyu/ccass-alert.git
```

## Usage

```bash
python alert.py
```
